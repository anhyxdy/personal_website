## 什么是 Agent？

Agent（智能体）是能自主感知环境、做出决策并执行行动的 AI 程序。

## 核心能力

- **感知** — 理解用户输入和环境状态
- **规划** — 拆解任务、制定步骤
- **执行** — 调用工具、API 完成操作

## 常见应用

- 代码自动修复 Agent
- 客服对话 Agent
- 自动化测试 Agent

> Agent 是 LLM 落地的重要方向，未来会有更多应用场景。
