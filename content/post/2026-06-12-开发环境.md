## 1. 包管理器：winget

Windows 11 自带 winget，一条命令搞定软件安装：

```powershell
# 搜索软件
winget search git

# 安装
winget install Git.Git
winget install Microsoft.PowerShell
winget install Microsoft.VisualStudioCode
```

## 2. 终端美化

### 安装 Oh My Posh

```powershell
winget install JanDeDobbeleer.OhMyPosh
```

### 配置 PowerShell Profile

```powershell
# 编辑 profile
notepad $PROFILE

# 添加以下内容
oh-my-posh init pwsh | Invoke-Expression
```

### 安装 Nerd Font

推荐 **JetBrainsMono Nerd Font**，在 Oh My Posh 官网下载安装。

## 3. 开发工具清单

| 工具 | 用途 | 安装命令 |
|------|------|---------|
| Git | 版本控制 | `winget install Git.Git` |
| Node.js | JS 运行时 | `winget install OpenJS.NodeJS` |
| Python | 脚本语言 | `winget install Python.Python` |
| VS Code | 编辑器 | `winget install Microsoft.VisualStudioCode` |
| Docker | 容器 | `winget install Docker.DockerDesktop` |

## 4. Git 配置

```bash
git config --global user.name "Your Name"
git config --global user.email "your@email.com"
git config --global init.defaultBranch main
```

## 5. VS Code 推荐插件

- **Prettier** — 代码格式化
- **ESLint** — JS/TS 代码检查
- **GitLens** — Git 增强
- **Material Icon Theme** — 文件图标
- **One Dark Pro** — 主题
