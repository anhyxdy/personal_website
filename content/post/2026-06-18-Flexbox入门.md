## 容器属性

```css
.container {
    display: flex;
    justify-content: center;  /* 水平居中 */
    align-items: center;      /* 垂直居中 */
    gap: 16px;                /* 间距 */
}
```

## 项目属性

```css
.item {
    flex: 1;           /* 等分剩余空间 */
    align-self: flex-end;  /* 单独对齐 */
}
```

Flexbox 是目前最实用的布局方案之一，一学就会 👍
